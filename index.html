Skinny WAR Maven Plugin

The Skinny WAR Maven Plugin repackages the WAR files in an EAR file into skinny WARs.

When no configuration values are set, the plugin sets default values.

Goals Overview

The Skinny WAR Maven Plugin has only one goal:

  • skinny-war-maven-plugin:repackage repackages the WARs in an EAR into skinny WARs.

Usage

General instructions on how to use the Skinny WAR Maven Plugin can be found on the usage page.

If you feel like the plugin is missing a feature or has a defect, you can fill a feature request or bug report in our issue tracker or submit a pull request.
